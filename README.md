# SAMURAI BOT V7

## FOR TERMUX USER

```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install nodejs-lts
> pkg install ffmpeg
> pkg install imagemagick
> npm i imgbb-uploader
> git clone https://github.com/GOLIATH130/SAMURAI-V7

> npm i

```

> git clone https://github.com/GOLIATH130/SAMURAI-V7
